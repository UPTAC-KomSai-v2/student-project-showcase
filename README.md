# UPTAC KomSai Student Project Showcase

A public showcase of selected student projects, coursework outputs, research prototypes, and capstone systems related to Computer Science courses.

This repository contains the GitHub Pages site for presenting project screenshots, videos, descriptions, and demonstrations developed by students under Computer Science classes and mentorship at the University of the Philippines Tacloban College.

---

## Purpose

This showcase aims to:

- Present selected student outputs publicly
- Highlight academic and technical projects
- Preserve demonstrations of student work
- Showcase practical applications of Computer Science concepts
- Provide portfolio visibility for students

---

## Contents

The showcase may include projects related to:

- Web Development
- Mobile Applications
- Artificial Intelligence and Machine Learning
- Data Science
- Database Systems
- Software Engineering
- Networking and Cybersecurity
- Embedded Systems and IoT
- Research and Thesis Projects

---

## Structure

Project media assets are organized by course.

```text

```
